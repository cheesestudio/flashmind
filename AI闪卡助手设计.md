# AI 闪卡助手 - 轻量版设计

## 项目概述

- **产品名**：AI 闪卡大师 / FlashMind
- **类型**：网页应用（单页应用）
- **核心功能**：AI 自动生成闪卡 + 智能复习计划
- **目标用户**：学生、职场人士、语言学习者

## 功能设计

### 核心功能

1. **AI 闪卡生成**
   - 粘贴文本/上传 TXT/MD 文件
   - 调用 AI API 生成问答卡片
   - 支持自定义卡片（手动添加）

2. **学习模式**
   - 翻转卡片看答案
   - 标记"记住" / "忘记"
   - 基于艾宾浩斯遗忘曲线的复习提醒（浏览器本地存储）

3. **卡片管理**
   - 创建/编辑/删除卡库
   - 导入/导出卡片（JSON 格式）

4. **多语言支持**
   - 中文、英文界面

### 数据存储

- **方案**：浏览器 localStorage
- **备份**：导出 JSON 文件

## 技术架构

| 层级 | 技术 |
|-----|------|
| 前端框架 | React + TypeScript |
| UI 组件 | Tailwind CSS / Radix UI |
| AI 接口 | OpenAI API / Claude API |
| 部署 | Vercel（免费） |
